# recruitment-crm
Recruitment Pipeline CRM is a web-based application developed using HTML, CSS, JavaScript, Node.js, Express.js, and SQLite. The system helps recruiters manage candidates through different recruitment stages such as Applied, Interview, Selected, and Rejected. It provides candidate management, status tracking, and dashboard analytics for efficient recruitment workflow management.

Project Name:

Recruitment Pipeline CRM

Description:

Recruitment Pipeline CRM is a web-based application that helps recruiters manage the hiring process efficiently. Users can add candidate information, view all candidates, update recruitment status (Applied, Interview, Selected, Rejected), delete records, and monitor hiring progress through a dashboard. The project is built with a simple and responsive user interface.

Features:

Add new candidate details
View all candidates in a table
Update candidate recruitment status
Delete candidate records
Dashboard showing recruitment statistics
Data stored using SQLite database
Responsive UI using Bootstrap
REST API using Express.js
